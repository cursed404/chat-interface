# Чат-приложение

Реализация интерфейса чата с возможностью:
- Отправки сообщений
- Динамическим добавлением участников
- Удалением участников
- Прокруткой контента

## Технологии
- React + TypeScript
- Vite
- Material-UI
- LESS
- Docker

## Установка
```bash
git clone https://github.com/cursed404/chat-interface.git
cd chat-interface
npm install

## Запуск
npm run dev
npm run build

## Docker
docker-compose up --build
